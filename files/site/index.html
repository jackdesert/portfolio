JackListed (work-in-progress, started Dec 2013)
A canonical, publicly-editable list of employers in the Internet industry
There's something intriguing about creating something that can be edited by anyone, without requiring a login. The barrier to entry is only millimeters high, yet you must find creative ways to prevent spam and fraud.
  • Rails 3.2 / Postgres / Nginx on Rackspace
  • Nginx
  • Foundation 5
  • Rspec/RR/Shoulda/FactoryGirl
  • Capistrano 3
  • Anyone can add listings
  • Reponsive, clean, interaction design requires few mouseclicks
  • City subdomains
  • Requires new URIs to have an http status of 200 (reduces spam and typos)
  • Requires new URIs to have a unique domain name (reduces spam and clutter)
  • Requires new URIs to contain certain keywords or else awaits moderator approval (reduces spam)

BreadnButter (2013)
The Simplest Way to add Breadcrumbs to Your Site
While other solutions for breadcrumbs require you to specify explicitly what text to display, what model to link to, etc., BreadnButter assumes RESTful routes and figures out the rest by itself.
  • A gem for Rails 3.1+ applications
  • Rails Engine to acquire gem's view files
  • Rspec tested
  • Add to your project via Gemfile
  • Insert into application.html.erb with one method call
